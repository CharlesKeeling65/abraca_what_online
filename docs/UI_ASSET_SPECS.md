# UI 资源占位与尺寸规范（Web / Mobile / Desktop）

> 当前版本使用占位块，后续可按本规范替换真实美术资源。

## 1. 通用牌面资源

| 资源 | 尺寸(px) | 格式 | 说明 |
| --- | --- | --- | --- |
| 咒语石正面 | 512 x 512 | PNG/WebP | 8 种法术统一模板，支持数字/图标叠层。 |

## 2. 桌面与背景

| 平台 | 资源 | 推荐尺寸(px) | 说明 |
| --- | --- | --- | --- |
| Desktop | 对局桌面背景 | 1920 x 1080 | 16:9 主背景，可扩展到 2560x1440。 |
| Web 通用 | 对局桌面背景 | 1600 x 900 | 兼容主流浏览器窗口。 |
| Mobile 竖屏 | 对局桌面背景 | 1080 x 1920 | 9:16，留安全区。 |
| Mobile 横屏 | 对局桌面背景 | 1920 x 1080 | 平板或横屏模式。 |

## 3. UI 组件贴图

| 资源 | Desktop/Web | Mobile | 说明 |
| --- | --- | --- | --- |
| 玩家头像框 | 256 x 256 | 192 x 192 | 显示昵称/血量/分数。 |
| 按钮底图 | 320 x 96 | 280 x 88 | 支持普通/悬停/禁用状态。 |
| 计分塔图层 | 1024 x 2048 | 768 x 1536 | 可拆成底图+指示物。 |
| 指示物（生命/分数） | 128 x 128 | 96 x 96 | 建议透明背景。 |

## 4. 导出建议

- 色彩空间：sRGB。
- 移动端额外提供 2x、3x 切图。
- 对高频 UI 图标优先使用 SVG。
- 文件命名建议：`{category}_{name}_{size}@{scale}.{ext}`。
